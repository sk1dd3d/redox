## ⚡ Quick Start

### 🔧 Installation

1. **Download the latest release**
   ```bash
   # Option 1: Direct download
   wget https://github.com/sk1dd3d/redox/releases/latest/download/redox-client.jar
   
   # Option 2: Using curl
   curl -L -o redox-client.jar https://github.com/sk1dd3d/redox/releases/latest/download/redox-client.jar

   # Option 3: Releases
   Go to the releases page.
   ```

2. **Install for your launcher**
   
   **For Minecraft Launcher:**
   - Place `redox-client.jar` in your `.minecraft/mods/` folder
   - Ensure Fabric/Forge is installed (version requirements in releases)
   
   **For MultiMC/PolyMC/Prism Launcher:**
   - Add as external mod through launcher interface
   - Configure Java arguments if needed
   - Please do not use TLauncher.

3. **First Launch Setup**
   - Launch Minecraft with Redox installed
   - Press `Right Shift` to open the client menu
   - Configure your preferred settings
   - Join a server and enjoy!

### 🎮 Quick Controls

| Key Combination | Action | Description |
|----------------|--------|-------------|
| `Right Shift` | Open Menu | Main client interface |
| `R + Right Shift` | Module List | Toggle module browser |
| `Tab` | Toggle HUD | Show/hide overlay |
| `F3 + R` | Reload Config | Refresh all settings |
| `Ctrl + Alt + L` | Panic Mode | Disable all modules instantly |
